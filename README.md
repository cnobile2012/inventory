A Django Implementation of a DIY Inventory Application
======================================================

This is updated code of the original `master` branch now renamed to `original`.
Most of the old Python 2 code has been updated to Python 3 code and Django is
also updated.

This is a single user inventory application, in other words it will not support
multiple inventory databases for different people. I had been working on a
version that would support multiple teams and users, however, that code is old
now also. I may in the future try doing this again, maybe reusing some of my
old code.
